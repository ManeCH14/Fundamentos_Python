Sistema de Descuentos en Python
Este programa permite calcular descuentos en compras según el monto y si el cliente es miembro de la tienda.

Funcionalidades:
-Aplica un 10% de descuento si el cliente es miembro y su compra supera el monto mínimo ($1000)
-Aplica un 5% de descuento si el cliente es miembro y su compra es menor al monto mínimo
-No aplica descuento si el cliente no es miembro
-Muestra el detalle de la compra, el descuento y el monto final

Conceptos aplicados:
-Condicionales (if, elif, else)
-Operaciones matemáticas básicas
-Manejo de cadenas (strip(), lower())
-Interacción con el usuario (input y print)
