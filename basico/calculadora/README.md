Calculadora en Python

Este programa es una calculadora básica desarrollada en Python que permite realizar operaciones matemáticas simples desde la consola.

Funcionalidades:
1.- Suma
2.- Resta
3.- Multiplicación
4.- División
5.- Validación de errores (entrada inválida y división entre cero)


Conceptos aplicados
-Estructuras de control (if, while)
-Manejo de excepciones (try/except)
-Entrada y salida de datos
