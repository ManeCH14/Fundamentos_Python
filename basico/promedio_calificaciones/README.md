Promedio de Calificaciones en Python

Este programa permite calcular el promedio de un conjunto de calificaciones ingresadas por el usuario desde la consola.

Funcionalidades:

-Solicita el número de calificaciones a evaluar.
-Permite ingresar cada calificación.
-Calcula el promedio automáticamente.
-Muestra todas las calificaciones ingresadas.
-Validación básica de errores.

Conceptos aplicados:

-Listas en Python.
-Ciclos (for).
-Entrada y salida de datos.
-Validación de datos.
-Operaciones matemáticas..

Este proyecto forma parte del aprendizaje de los fundamentos de Python y busca reforzar el uso de estructuras básicas del lenguaje.
