Juego: Adivina el Número
Este programa es un juego en Python donde el usuario debe adivinar un número secreto generado aleatoriamente. El jugador tiene un número limitado de intentos y recibe pistas sobre si el número ingresado es mayor o menor que el número secreto.

Funcionalidades:
-Genera un número aleatorio entre 1 y 50
-Permite adivinar el número con 5 intentos
-Indica si el número ingresado es mayor o menor que el número secreto
-Informa cuántos intentos quedan
-Termina el juego cuando se adivina el número o se acaban los intentos
